So you want to make your own guide?

How to start

For starters, you'll need a Github account and git installed in your computer. With this, you should be able to create a fork of the repo for you to modify.

With your own copy of the repo, it's time for you to clone it to your computer.

And now you need to just download the repo to your computer:

git clone https://github.com/Furitenman/mjg-repo.git

And that's it! You can start working on the repo.

Learn more about git.

Then, you should be familiar with Bootstrap. It's a library that does pretty much all the hard work in terms of css and html. You just need to invoke classes accordingly to your needs.

Read more about Bootstrap.

What to modify

The easiest thing to do with this template is just modify this structure and add content. Check all the <h1></h1> tags for their id, which you may want to modify to something more accurate to what you want to section. Then modify the navbar's links according to this ids, which will help others to quickly surf through the pages.

Add to the repo

You got your guide ready? Now it's time to push to the repo.

git add mynewguide.html
git commit -m 'Check out this new guide'
git push repository master
